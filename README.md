# Environment

# Installation

```bash
composer require biiiiiigmonster/laravel-cleanable
```

# Introductions
删除作为数据操作的生命周期最后一环，受到关注度较小，然而在业务中完整数据的关联性往往会因为这些疏忽而被破坏。
这个包可以很方便的帮您管理这些关联数据删除关系，仅仅只需要简单的定义。
让我们来尝试一下吧！
